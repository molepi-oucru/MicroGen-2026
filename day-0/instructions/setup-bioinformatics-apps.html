0.1.4 Optional graphical bioinformatics apps

Author

Minh-Quan Ton-Ngoc

Published

May 11, 2026

1 Optional graphical bioinformatics apps

These apps are optional. Install them only if your instructor says they will be used in a local activity. Many workshop activities will use the cloud server instead.

1.1 Artemis

Artemis is a Java-based genome browser and annotation tool. Install OpenJDK 17 first, then run Artemis from its .jar file.

Official pages:

  • Artemis website: https://sanger-pathogens.github.io/Artemis/Artemis/
  • Artemis GitHub releases: https://github.com/sanger-pathogens/Artemis/releases

1.1.1 Windows

  1. Install Temurin OpenJDK 17 LTS using the Windows instructions above.
  2. Download Artemis from the official Artemis page or GitHub releases.
  3. If you downloaded a ZIP file, extract it to a simple folder, for example:
C:\Users\YourName\Downloads\Artemis
  1. Open PowerShell.
  2. Move into the Artemis folder. Replace the path with the folder where you extracted Artemis:
cd "C:\Users\YourName\Downloads\Artemis"
  1. List the .jar files:
dir *.jar
  1. Run Artemis from the command line. Replace filename.jar with the real file name you see:
java -jar filename.jar

For example, if the file is called artemis.jar, run:

java -jar artemis.jar
Why use the command line?

Running Artemis with java -jar filename.jar makes Java errors visible in the terminal. This is easier to troubleshoot than double-clicking a .jar file.

1.1.2 macOS

  1. Install Temurin OpenJDK 17 LTS using the macOS instructions above.
  2. Download Artemis from the official Artemis page or GitHub releases.
  3. If the download is a ZIP file, double-click it to extract it.
  4. Open Terminal.
  5. Move into the Artemis folder. Replace the path with your real folder:
cd ~/Downloads/Artemis
  1. List the .jar files:
ls *.jar
  1. Run Artemis. Replace filename.jar with the real file name:
java -jar filename.jar

If macOS blocks the app or downloaded files, see the Gatekeeper troubleshooting section below.

1.1.3 Ubuntu

Use this only on your own Ubuntu computer, VM, or WSL setup with graphical support. Do not install system packages on the shared workshop server unless an instructor asks you to.

sudo apt update
sudo apt install openjdk-17-jdk artemis
java -version
art

If art does not start Artemis on your Ubuntu version, try:

artemis

If the Ubuntu package is unavailable or too old, install OpenJDK 17 and run the Artemis .jar file manually:

java -jar filename.jar

1.2 BandageNG

BandageNG is a graphical application for viewing de novo assembly graphs. This guide uses BandageNG instead of the older Bandage application.

Official pages:

  • BandageNG GitHub: https://github.com/asl/BandageNG
  • BandageNG Bioconda package: https://bioconda.github.io/recipes/bandage_ng/README.html
BLAST+ may be needed for local BandageNG activities

Install NCBI BLAST+ if you plan to run BandageNG on your own computer during sessions that use BLAST-related BandageNG features. If you only view simple graph files without BLAST searches, BLAST+ may not be required.

1.2.1 Install BandageNG on Windows

  1. Open the BandageNG GitHub page: https://github.com/asl/BandageNG
  2. Open the Releases page.
  3. Download a Windows release if one is available for your workshop version.
  4. Extract the archive.
  5. Run the BandageNG executable from the extracted folder.

If no Windows binary is available for the current BandageNG release, use the workshop server or the Ubuntu/conda option below.

1.2.2 Install BandageNG on macOS

  1. Open the BandageNG GitHub page: https://github.com/asl/BandageNG
  2. Open the Releases page.
  3. Download the macOS build for your chip if one is available:
    • Intel Mac: x86_64 or x64
    • Apple Silicon Mac: arm64 or aarch64
  4. Extract the archive and open BandageNG.

If macOS blocks the app, see the Gatekeeper troubleshooting section below.

1.2.3 Install BandageNG on Ubuntu

The recommended Ubuntu route is to use a conda-compatible package manager such as conda, mamba, micromamba, or pixi:

conda create -n bandageng -c conda-forge -c bioconda bandage_ng
conda activate bandageng
BandageNG

If the executable name differs in your environment, list matching commands:

compgen -c | grep -i bandage

1.2.4 Install NCBI BLAST+ for BandageNG

Official NCBI BLAST+ download page:

https://ftp.ncbi.nlm.nih.gov/blast/executables/blast+/LATEST/

1.2.4.1 Windows

  1. Open the NCBI BLAST+ latest downloads page.
  2. Download the Windows 64-bit BLAST+ installer or archive. The filename usually contains win64.
  3. Run the installer or extract the archive.
  4. If you used an archive, add the BLAST+ bin folder to your Windows PATH.
  5. Open a new PowerShell window and check:
blastn -version
makeblastdb -version

1.2.4.2 macOS

Option A: official NCBI archive:

  1. Open the NCBI BLAST+ latest downloads page.
  2. Download the macOS archive matching your system if available.
  3. Extract it.
  4. Add the extracted bin folder to your PATH.
  5. Check:
blastn -version
makeblastdb -version

Option B: Homebrew, for users who already use Homebrew:

brew install blast
blastn -version
makeblastdb -version

1.2.4.3 Ubuntu

Install the Ubuntu package:

sudo apt update
sudo apt install ncbi-blast+
blastn -version
makeblastdb -version

1.3 SeaView 5

SeaView is a graphical tool for multiple sequence alignment and molecular phylogeny.

Official page:

https://doua.prabi.fr/software/seaview/

1.3.1 Windows

  1. Open the SeaView official page.
  2. Download the Windows seaview5.exe file.
  3. Run seaview5.exe.
  4. The file is a self-extracting archive. Choose where to create the seaview5 folder.
  5. Open the new seaview5 folder and run the SeaView program.

The official SeaView page notes that the Windows download creates a folder containing SeaView, example data, help files, and helper programs such as muscle, clustalo, phyml, Gblocks, and treerecs.

1.3.2 macOS

  1. Open the SeaView official page.
  2. Download the macOS SeaView archive.
  3. Extract the ZIP file.
  4. If macOS blocks the app the first time, Control-click the SeaView icon and choose Open.

If the app is still blocked, see the Gatekeeper troubleshooting section below.

1.3.3 Ubuntu

Use the Ubuntu package on your own computer, VM, or WSL setup with graphical support:

sudo apt update
sudo apt install seaview
seaview

SeaView is graphical. If seaview starts but no window appears, check that you are using a normal Ubuntu desktop session, X11 forwarding, or VNC.

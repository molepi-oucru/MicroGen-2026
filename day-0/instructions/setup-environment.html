0.1 Setup your workspace

Author

Minh-Quan Ton-Ngoc

Published

May 11, 2026

1 Purpose

This guide helps you prepare a Windows or macOS laptop for the MicroGen 2026 workshop. It covers the tools used for coding, data analysis, and connecting to the workshop cloud server.

Most heavy bioinformatics workflows will run on the workshop server, not on your laptop. Your laptop mainly needs:

  • a code editor,
  • R and Python for small local exercises,
  • SSH tools for connecting to the server,
  • and optional graphical tools for users who prefer them.
Required and optional tools

Required for most participants: Python 3.13, R 4.5.0, Visual Studio Code, the VS Code Remote - SSH extension, and the listed Python/R packages.

Optional: RStudio Desktop, Java, Artemis, BandageNG, SeaView, and extra X11/VNC tools. Instructors may ask you to use these for specific graphical activities.

Do not share secrets

If you receive a .pem private key file, do not upload it, email it, paste it into chat, or share it with another participant. The key is your login credential for the server.

3 Which instructions should I follow?

  • If you use Windows 10 or Windows 11, follow Windows 10/11 setup.
  • If you use an Intel Mac, follow macOS setup, choosing the Intel options.
  • If you use an Apple Silicon Mac with an M1, M2, M3, M4, or newer Apple chip, follow the Apple Silicon options.
  • If you use Ubuntu on your own laptop, WSL with graphical support, or an Ubuntu virtual machine, follow Ubuntu setup.
How to check your Mac chip

Open the Apple menu in the top-left corner, then choose About This Mac.

  • If you see Processor: Intel, use the Intel Mac instructions.
  • If you see Chip: Apple M1, M2, M3, M4, or similar, use the Apple Silicon instructions.
